# Errors

Project-local failures worth remembering.

---

## [ERR-YYYYMMDD-001] short-title

- Logged: ISO-8601 timestamp
- Area: docs | tests | backend | frontend | infra | workflow
- Summary: One concise sentence
- Error: Redacted error snippet or short description
- Context: What was being attempted
- Likely Fix: Short next-step suggestion
- Promotion: none | skill-candidate | rule-candidate
